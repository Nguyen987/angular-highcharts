This library aims to provide a lightweight wrapper around Highcharts.

Consult [the demo](demo/usage-demo.component.js) for usage example.


### Setup
Clone project repo, and inside its directory, execute:
```
npm install
```

### Run unit tests
```
npm test
```

### Run the dev task
```
npm run watch
```

### Dependencies
+ Angular 1.5+
+ Highcharts 4+
